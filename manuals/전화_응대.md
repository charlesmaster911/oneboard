# 📞 전화 응대 매뉴얼

> **⛔ 절대 금지:** 상대가 누구인지 · 왜 전화했는지 파악 못한 상태에서 **약속·일정 확정 금지**.
> 상대 정보를 메모지에 적은 뒤 대응합니다.

---

## 🟡 황금 3문장 — 무조건 먼저 물어본다

전화를 받으면 **3문장을 순서대로** 확인합니다. 상대가 누구이고, 어디에서, 왜 전화했는지 파악되기 전까지 어떠한 약속도 하지 않습니다.

1. **"실례지만 성함이 어떻게 되세요?"**
2. **"어느 회사·어디에서 연락 주셨나요?"**
3. **"어떤 일로 연락 주셨을까요?"**

---

## 🔀 전화 유형별 분기 — 황금 3문장 답변 듣고 판단

### 유형 A · 세일즈 (광고·영업·마케팅 제안)

**신호:** "마케팅 제안", "광고 대행", "홈페이지 제작", "서버 호스팅", "보험"

**대응 스크립트:**
> "담당자가 자리에 없습니다. 제안서는 **공식 메일**로 보내주시면 검토 후 연락드리겠습니다."

→ **약속 잡지 않고 이메일로 유도**. 담당자 전달 불필요.

---

### 유형 B · B2B 문의 (업장·호텔·카페 등 사업자)

**신호:** "바", "호텔", "레스토랑", "납품", "견적", "도매"

> ⚠️ **필수 분기 질문:**
> "얼음이 필요하신 건가요, 얼음 **제조기(메이커)**가 필요하신 건가요?"

**5항목 메모:** 용도 · 수량 · 지역 · 예산 · 납기

**대응 스크립트:**
> "확인해드리고 **담당자가 1시간 안에 다시 연락드리겠습니다**. 연락 가능한 번호·이메일 부탁드립니다."

→ 박지현 팀장 이관.

---

### 유형 C · 대표 찾음 ("대표님 계신가요?")

**신호:** "대표", "사장", "김진수"

**황금 3문장 먼저 확인** (누구·어디·왜). 파악 전 연결 불가.

**대응 스크립트:**
> "대표님은 외부 미팅 중이십니다. 용건 전달해드리고 회신 일정 잡아드리겠습니다. 성함·연락처·용건 말씀해 주시겠어요?"

→ 상대·용건 파악 완료 후 **찰스에게 텍스트로 전달**. 직접 연결 원칙 금지.

---

### 유형 D · 기존 거래처

**신호:** 상대가 본인·회사 먼저 밝힘, 이름이 거래처 리스트에 있음

**대응 스크립트:**
> "네 안녕하세요 OO님! 담당자 연결해드릴게요 / 지금 자리에 없어서 메시지 남겨드릴게요."

→ 담당자 바로 연결 또는 메시지 받아 전달.

## 📝 전화 메모 양식 — 전화 받을 때 항상 옆에

| 항목 | 내용 |
|------|------|
| ① 시간 | 받은 시각 |
| ② 성함·직함 | 이름 + 직함 |
| ③ 회사·어디 | 회사명 또는 매장명 |
| ④ 연락처 | 전화번호 + 이메일 |
| ⑤ 용건 | 한 문장으로 요약 |
| ⑥ **B2B면 얼음/메이커** | 얼음 구매? 메이커 구매? |
| ⑦ 다음 액션 | 누가·언제 회신 |

## 🤝 B2B·대량 리드 → 박지현 팀장 이관 절차

전화 접수 단계에서 **5항목** 먼저 수집한 뒤 팀장에게 넘긴다. **현장에서 가격·납기 확정 금지.**

```
① 용도 → ② 수량 → ③ 지역 → ④ 예산 → ⑤ 납기
```

### 이관 기준
- **10개 이상 대량·법인·각인 문의** → 팀장 이관 (1차 가격 자율 -10% 권한)
- **1회 2천만 원 이상 단일 거래 가능성** → 찰스 INBOX 에스컬레이션
- **언론·미디어 문의** → 즉시 찰스 전달

## 📌 핵심 원칙 (3줄 요약)

2. **현장에서 가격·납기 확정 금지** — 담당자 회신 원칙
3. **메모 양식에 빠짐없이 기록** — 7개 항목
