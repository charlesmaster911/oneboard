## 1차 영상 상세 기획 — 각 구성품 소개 및 사용법

### 영상 제목: "아이스 몰드 사용 설명서"

### 1. 오프닝
- 제품 로고 + 짧은 인트로
- 내레이션: *"신선하고 깨끗한 얼음을 집에서! 간편하게 만드는 방법을 알려드립니다."*

### 2. 구성품 소개 (3종)
| 구성품 | 설명 문구 |
|---|---|
| **케이스** | "단단하고 투명한 케이스가 얼음을 보호합니다." |
| **물통** | "물을 담는 튼튼한 물통으로 누수 걱정 없이 사용 가능합니다." |
| **몰드** | "얼음을 형성하는 몰드는 깔끔하고 빠른 분리가 가능합니다." |

- 일상 물품(책·스마트폰) 옆에 놓고 크기 비교
- 결합 전후 비교 샷

### 3. 얼리기 전 사용 방법
1. 순서대로 결합: 몰드 → 케이스 → 물통
2. 물통 외부 물기 제거 (마른 수건으로 닦기)
3. 따뜻한 물 사용 — **60도 정수기 따뜻한 물 기준** (온도계 활용)
4. 차가운 물 사용 금지 (X표시)
5. 몰드 결합 후 위쪽에서 물 추가 채우기
6. 냉동고 바람 나오는 곳 피하기 (최대한 멀리 배치)
7. 너무 낮은 온도 냉동고 사용 금지 (X표시)

### 4. 얼린 후 사용 방법
1. 케이스와 물통 분리
2. 물통 외부에 따뜻한 물 붓기 (쩌적 소리 포함)
3. 물통을 옆으로 눕히고 한 손은 물통·한 손은 몰드 손잡이 잡고 양쪽으로 눌러 분리
4. 몰드를 비틀면서 얼음 탈거
5. 몰드 갈라주고 빵칼·과도·숟가락 등으로 몰드 사이 표면에 넓게 얼어진 얼음 제거
6. 얼음이 낀 몰드 양쪽 끝 잡고 비틀기
7. 물통 하단 잔여 얼음은 물통을 뒤집고 뜨거운 물 부어 약 1분 후 툭툭 쳐서 제거
8. 얼음 원하는 크기로 잘라 사용

### 5. 클로징
- 내레이션: *"깔끔하고 신선한 얼음으로 여러분의 일상에 시원함을 더하세요!"*
- 제품 로고 + 문의 사항 + 웹사이트 정보

## 2차 영상 — AS 문의 / 궁금사항

### 다룰 주제 10+
1. **세척 방법**
   - 온수로 몰드·물통(하얀가루) 세척
   - 방한케이스는 내부 건조 + 외부는 물 묻힌 거즈/행주로 닦기
   - 충분히 세척·건조하면 고무·실리콘 냄새 제거
2. **얼음 보관 방법** — 서리 낀 얼음 → 음료에 넣으면 투명, 지퍼백·락앤락통 보관
3. **EPP 케이스 끼는 현상** — 얼리면 물통 팽창하며 공간 늘어남
4. **눕혀 사용 불가** — 밀봉 제품 아니라 물 새어나옴
5. **벌어지는 손잡이** — 얼음 품질에 영향 없음 (고의 벌린 후 얼린 얼음 비교 샷)
6. **하얀 가루** — 물에 녹아있던 미네랄 고체화 → 온수 세척으로 제거
7. **스틱 형태 물통 끼임** — 미니 스틱 형태 유지 때문에 벽면 맞닿도록 제작
8. **실리콘/TPR 고무 냄새** — 주방세제 세척 후 상온 2~3일 건조
9. **몰드 홈 불일치** — 얼음 완성도에 문제 없음 (테스트 완료)
10. **아이스볼 타원 찌그러짐** — 실리콘으로 교환 AS
