# SOP · B2B 문의 접수 프로세스

**원본:** `jewelice-playbook/jihyun-unified.html` (SOP 3번)
**책임자:** 박지현 팀장 (루시)

---

## 🔀 7단계 프로세스

### 1. 문의 접수 → 기본 정보 확인
- 업체 규모
- 발주량
- 납기 (정기·일회성)
- 담당자 연락처 (전화·이메일)

### 2. 1차 가격 제안 (당일 회신)
- **자율 할인 권한: -10%까지**
- 당일 내 1차 견적 발송
- 제품 구성 + 수량 + 예상 납기 포함

### 3. 재고·생산 확인
- 현재 재고 확인
- 생산 스케줄 조정 필요 여부
- 배송 권역·방식 확인

### 4. 상한 초과 시 찰스 INBOX 에스컬레이션
- **1회 2천만 원 이상 단일 거래**
- 자율 권한(-10%) 초과 할인 요청
- 특수 조건·커스텀 각인 대량
- 언론·미디어·공공기관 관련

→ `MISSION/INBOX.md` 기재 + 찰스 승인 대기

### 5. 찰스 승인
- 찰스 확인·조건 조율
- 승인 시 정식 견적서 발행

### 6. 견적서 발행 → 계약
- `jewelice-playbook/jihyun-berish-quote.html` 템플릿 사용
- 계약서명은 **찰스**가 최종 서명
- 선불·월정산 결제 조건 명시

### 7. 납품 후 실적 입력
- 이지어드민 B2B 발주 기록
- 매출 KPI 대시보드 갱신
- 팔로업: 납품 후 1주 내 만족도 확인 메일

---

## 🚨 긴급 에스컬레이션 5조건 (즉시 찰스 알림)

1. 매출 -20% 이탈 조짐
2. 팀원 퇴사 통보
3. 법적 이슈 (제품 하자·클레임·분쟁)
4. 단일 거래 2천만 원 이상
5. 언론 노출 가능성
