## 📦 택배 발송 SOP (5단계)

### 1. 오전 9시 · 전날 주문 취합
- 이지어드민 로그인 → 전날 23:59까지 주문 전체 export
- 결제 완료 + 미발송 건만 필터링

### 2. 포장 완료 후 송장 출력
- 제품별 포장 (듀오메이커 / 아블러 / 스피어메이커 / 악세서리)
- 이지어드민에서 송장 일괄 출력
- 송장 부착 후 2차 확인 (수량·주소·품목)

### 3. 대한통운 접수
- 지정 시간까지 택배사 접수
- 접수증 보관

### 4. 발송 완료 후 주문 상태 업데이트
- 이지어드민 상태: "발송 완료"로 일괄 변경
- 스마트스토어·쿠팡 연동 자동 갱신 확인

### 5. CS 이슈 발생 시 즉시 고객 연락
- 파손·지연·오배송 발생 시 **고객에게 먼저 연락** (질문 받기 전에)
- 팀장·찰스 보고 → 대응 방침 확정 후 회신

## 🚨 택배 CS 이슈 4분기

### A. 배송 지연
- 사유 확인 (택배사 접수 지연·주소 문제·폭주)
- 고객에 **새 도착 예정일** 선제 안내
- 사과 + 운송장 링크 + 재확인 일정

### B. 파손
- 택배사 접수 번호 확보
- 고객에 사진 요청 + 주문번호 기록
- 팀장 보고 → 교환·환불 결정
- 불량 30일 이내 무상 처리 규정 적용

### C. 오배송
- 회수 일정 확정 (고객 번거로움 최소화)
- 재발송 일정 선확정 후 공유
- 왕복 배송비 **쥬얼아이스 부담**

### D. 분실
- 택배사 조회 + 수령인 확인 (경비실·무인택배함)
- 3영업일 미확인 시 재발송 결정
- 분실 건은 택배사 배상 별도 진행

## 🔗 매뉴얼 연결

전화·채널톡으로 배송 문의 들어오면:
1. **team-manual.html** "🎧 고객 CS" 탭 → "유형 ① 배송" 카드 스크립트 사용
2. 주문번호 확인 → 본 SOP 5단계 기준으로 현황 확인
3. 이슈 유형(A~D) 판단 → 해당 분기 적용
