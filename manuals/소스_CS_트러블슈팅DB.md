## 카테고리별 실전 케이스

### 1. EPP 관련 (물통 이슈) — **빈발 불량 최상위**

> EPP = Expanded PolyPropylene. 메이커 얼음통을 감싸는 단열 폼. EPP와 내부 물통 결합부가 가장 취약한 파손·불량 지점.

#### 케이스 A: EPP-물통 끼는 현상 (8건)
- **증상**: 물통이 EPP에 꽉 끼어 분리 안 됨
- **제품**: 주로 스틱 화이트, 아이스볼 블랙/브라운, 스피어 블랙
- **대응**:
  1. 카카오채널/채널톡으로 증상 접수
  2. 회수 요청 → 며칠 내 입고 확인
  3. 교환 제품 출고 (사무실 또는 모두먼트)
- **패턴**: 특정 색상·모양 편중 아님. 생산 배치 전반 이슈.

#### 케이스 B: EPP-떠있는 현상 / EPP 분리 (7건)
- **증상**: 물통이 EPP에서 뜨거나 완전 분리됨
- **대응**: **맞교환 처리** (출고와 회수 동시 진행)
- **내부 메모**: "맞교환 상태 깐깐하게 확인해서 보내드려야 함"
- **리콜성 대응**: 반복 파손 시 다른 모양으로 교환 (예: 스틱 → 아이스볼)

#### 케이스 C: 반복적인 물통 파손 (2건)
- **증상**: 교환 후에도 계속 파손
- **대응**: 다른 모양으로 교환 출고 + 테스트 진행 (아이스볼 화이트 등)

### 7. 변심 반품/교환 (배송비 정책)

| 유형 | 배송비 | 절차 |
|------|-------|------|
| 변심 교환 | 6,000원 (왕복) / 제주 12,000원 | 입금 확인 → 수거 요청 → 교환 발주 |
| 변심 반품 | 왕복 배송비 차감 | 자동 수거 또는 회수 요청 → 환불 |
| 스마트스토어 주문실수 | 7,000원 (왕복) | 입금 확인 → 반품 처리 |
| 자사몰 주문실수 | 6,000원 (왕복) | 입금 확인 → 반품/교환 |

### 변심 교환 인기 전환 패턴

| 원 주문 | 교환 요청 | 빈도 |
|---------|---------|------|
| 큐브 → 아이스볼 | 다수 | 많음 |
| 스틱 → 아이스볼 | 다수 | 많음 |
| 아이스볼 → 스틱 | 있음 | 보통 |
| 큐브 → 스피어 | 있음 | 소수 |
| 큐브(화이트) → 다이아(화이트) | 있음 | 소수 |
| 큐브 3개 → 스틱 3개 | 있음 | 소수 |

> **인사이트:** **"큐브/스틱 → 아이스볼"이 가장 많은 변심 교환 방향**. 구매 단계에서 아이스볼을 디폴트 추천하면 CS 부담 감소 가능성.

## 회수/교환 프로세스 표준 흐름

```
[ CS 접수 ]
    ↓
[ 회수 요청 ] (카카오채널 등에서 안내)
    ↓
[ CNplus 반품 접수 ] (기업고객건별접수 → 반품 구분)
    ↓
[ 입고 확인 ] (평균 3~7일 소요)
    ↓
[ 제품 검수 ] (맞교환인 경우 사전 출고 가능)
    ↓
[ 교환 제품 출고 ] (사무실 출고 or 모두먼트 출고)
    ↓
[ CJ대한통운 송장번호 고객 전달 ]
    ↓
[ 완료 처리 ]
```

## 핵심 운영 인사이트 (5줄 요약)

1. **CS 유입의 42%가 카카오채널** — 카카오톡 1:1 응대 속도·품질이 전체 CS 품질의 핵심.
2. **EPP-물통 관련 불량이 구조적 취약점** — 끼는 현상 + 떠있는 현상 + 분리 = 총 15건 이상. 제조 공정 QC 필요.
3. **"스틱 블루 스티커 → 아이스볼 내용물" 오부착 19건** 동일 패턴 — **물류 포장 공정에서 특정 배치 전체 오부착** 의심. 1회성 아님.
4. **화이트 케이스 스크레치 반복** — 외관 QC 및 포장 방식 재검토 필요.
5. **변심 교환 주 패턴은 "큐브/스틱 → 아이스볼"** — 사전에 아이스볼을 디폴트 추천하는 상세페이지 개편이 CS 감축에 직결.
