## 1. 발주 업무

### 1-1. 이지어드민 & 이지오토
1. 이지오토 하단부 클릭
2. 자동발주 클릭
3. 창은 끄지 말고 다른 업무 이후 발주 종료 여부 확인

### 1-2. 이지어드민 (수동발주 필요 영역)
이지어드민 연동 안 된 곳은 수동발주:

**유튜브 쇼핑**
1. 체크 후 엑셀 다운로드
2. 송장 폴더에 당일 날짜로 생성하여 전일 폴더의 통합파일 사용
3. 유튜브 시트에 다운받은 파일 복사 붙여넣기
4. 피플스토리 시트에 적용됐는지 확인 (수식 확인 필요)
5. 정상 적용 시 피플스토리 폴더에 붙여넣고 해당 날짜로 저장

**피플스토리 업로드**
1. 파일 업로드 → 정상 업로드 확인
2. 다음 클릭 → 메모/매칭/합포 전체 진행
3. 오후 3시 이후 정상 배송 확인 후 수동 발주 건 송장번호 입력

## 2. 업무 관리 — 두 개 북마크 창 모두 열기

### 2-1. 판매자 페이지 (8개 채널)
1. 이지어드민 → 발주 및 재고 확인
2. 카페24 → 자사몰 & 유튜브
3. 네이버 스마트스토어 관리자 페이지
4. 쿠팡 → `hanbandokitchen`(듀오메이커) / `nemochip`(아블러) — **아이디 2개 각각 확인**
5. 카카오 → 톡스토어 / 선물하기 **두 곳 다 확인**
6. 와디즈
7. 오늘의집
8. 그 외 다수

### 2-2. 업무 관리 시트 (5종)
1. 쥬얼아이스 업무참고 엑셀 → CS 내용 정리 등
2. 플랫폼별 유입현황 → 자사몰·스마트스토어·쿠팡(2)·카카오(2) 합계 매출 및 유입수 입력
3. 쥬얼아이스 업체별 매출 보고서 → 각 몰 합계 금액 입력
4. 통합리포트 → 자사몰·메타·쿠팡·네이버·유튜브쇼핑·카카오 광고 및 매출 입력
5. 결제승인요청 → 결제 승인 내용 입력 (이전 기록이 어떤 제품을 발주했는지 확인용으로도 활용)

## 4. CS 응대 및 리뷰 관리 (4개 채널)

### 4-1. 채널톡 (문의 진입)
기본 응대 프로세스:
1. **구매 고객 CS**
   - 성함 + 전화번호 확인
   - 이지어드민 CS 창에서 주문 확인 가능
   - 모르는 내용은 과거 응대 내역 확인
   - 또는 "업무참고 엑셀 FAQ(수정)" 시트에서 검색하여 응대
2. **교환·반품 문의**
   - 사진으로 증상 확인 후 진행
   - 사진 확인 어려우면 사무실로 제품 받아서 직접 테스트 후 교환·반품 진행

### 4-2. 스마트스토어 (문의 + 리뷰)
- 민감한 사항(하자 등) 문의는 **카카오 채널로 문의 달라고 응답** 유도
- 리뷰는 제미(제미나이?) 자동 처리

### 4-3. 자사몰 (카페24) 리뷰
- 경로: 앱 → 마이앱 → 알파리뷰 관리하기 → 상단 "쥬얼아이스" 클릭
- 자동으로 리뷰 댓글 작성됨
- **민감한 리뷰는 답변 후 게시중지** 처리

### 4-4. 메타 (인스타그램·페이스북)
- 댓글 관리

## 5. 개선 과제 (후임자 인수 포인트)

1. **FAQ 및 답변 템플릿 정리 필요**
   - 현재 "업무참고 엑셀 FAQ(수정)" 시트에 흩어져 있음
   - 채널톡·스마트스토어·카페24 공통 답변 통합 템플릿화 권장

2. **민감 리뷰 escalation 기준 문서화**
   - 현재는 "민감하면 게시중지" 수준
   - 하자·환불·악성 리뷰 구분 기준, 보고 라인 명문화 필요

3. **쿠팡·카카오 아이디 이중 운영 체크리스트**
   - 쿠팡(hanbandokitchen·nemochip) + 카카오(톡스토어·선물하기) 각 2개씩 운영
   - 누락 방지 일간 체크리스트 필요

4. **피플스토리 송장 업로드 자동화**
   - 현재 수동 복붙 + 수식 확인 단계 많음
   - 이지어드민 연동 확대 또는 스크립트화 검토
